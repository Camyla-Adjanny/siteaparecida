Estrutura de um Documento HTML:

  • <!DOCTYPE html>: Declaração do tipo de Documento.
  • Tags <HEAD>: Informações como título e links para CSS/JS.
  • Tags <BODY>: Conteúdo da página, como textos e imagens.

O que são TAGS?

TAGS HTML informam ao navegador como formatar elementos como textos, links, imagens, etc. São mais de 100 tags disponíveis!
