Como é a Estrutura de um Documento HTML?

A estrutura básica de um documento HTML é composta por três partes: A declaração do tipo de Documento, o Cabeçalho e o Corpo.

Declaração do Tipo de Documento

  • A instrução <!DOCTYPE html> deve ser a primeira linha do documento.
  • Esta instrução informa ao navegador que o código é HTML.

Cabeçalho

  • Delimitado pelas tags <HEAD> e </HEAD>.
  • Contém informações sobre o documento, como o título, a codificação e links para arquivos CSS e JavaScript.

Corpo

  • Delimitado pelas tags <BODY> e </BODY>.
  • Contém conteúdo da página, como textos, imagens, formulários, vídeos, mapas, entre outros.

As tags HTML são formadas por uma tag de abertura, o conteúdo e uma tag de fechamento. A tag de abertura começa com o símbolo <> e a tag de fechamento termina com o símbolo </>.

O que são as TAGS, como identificá-las e como usá-las?

TAGS HTML são instruções que informam ao navegador como formatar o texto. Você pode usá-las para definir itálico, quebras de linha, objetos, bullet points e muito mais.

Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet. Ao todo, são mais de 100 tags que podem ser utilizadas!!
